# Food Delivery Analysis Dashboard

## Project Overview
100K+ food delivery orders analysis using Python, SQL, PowerBI.
**Business Impact**: Customer insights, delivery optimization, revenue growth.

## Tech Stack
- **Data Processing**: Python (Pandas, NumPy)
- **Database**: SQL 
- **Visualization**: PowerBI Dashboard
- **Tools**: VS Code, GitHub

## Folder Structure
online_food_delivery_analysis/
├── data_raw/ # Original CSV
├── data_processed/ # Cleaned + Features
├── notebooks/ # EDA + ML notebooks (4 files)
├── sql_scripts/ # Database tables.sql
├── powerbi/ # food_delivery_dashboard.pbix
├── docs/ # Screenshots + Insights
└── README.md


## Quick Start
1. Clone repo: `https://tamilarasirajendran.github.io/food-delivery-analysis/
2. Open PowerBI file: powerbi/food_delivery_dashboard.pbix
3. Run SQL: sql_scripts/tables.sql
4. Notebooks execute pannu Jupyter-ல.

## Key Insights
- Peak hours: 7-9 PM (45% orders)
- Top cuisine: Biryani (28%)
- Avg delivery time: 22 mins
- Revenue driver: Discounts + Loyalty
